# RockPaperScissorsGame

## Dataset

Dataset was downloaded from
[kaggle](https://www.kaggle.com/datasets/frtgnn/rock-paper-scissor) and shall
be extracted to `~/.kaggle/rock-paper-scissors`.

The following structure is assumed:
```
~/.kaggle
└─── rock-paper-scissors
    ├─── rps
    │   └─── rps
    │       ├─── paper
    │       ├─── rock
    │       └─── scissors
    └─── rps-test-set
        └─── rps-test-set
            ├─── paper
            ├─── rock
            └─── scissors
```
